# Colección de Tests Interactivos - Programación Java (IFCD0052PO)

Este repositorio contiene la batería de simulacros de exámenes y pruebas de autoevaluación desarrolladas para el certificado oficial **IFCD0052PO Programación Java**. 

El objetivo es proporcionar un entorno de evaluación portable, seguro y automatizado para el alumnado.

## 🚀 Acceso Directo (Simuladores Online)

Puedes realizar los simulacros de examen directamente en el navegador a través de GitHub Pages:

*   [📌 S38: Test de Repaso General](https://heipry.github.io/java-tests-IFCD0052PO/S38-Test-Repaso.html)
*   [📌 S39: Fundamentos Java Core](https://heipry.github.io/java-tests-IFCD0052PO/S39-Test_Fundamentos_Core.html)
*   [📌 S39: Simulacro de Examen B](https://heipry.github.io/java-tests-IFCD0052PO/S39-Test_SimulacroB.html)
*   [📌 Simulacro de Examen Final](https://heipry.github.io/java-tests-IFCD0052PO/Simulacro-examen-final.html)

*   [🚀 **Java Exam Automator**](https://github.com/heipry/corrector-examenes)  Es necesario para obtener notas y estadísticas instantáneas.

## 🛠️ Características del Sistema

Estos archivos HTML integran una lógica avanzada de gestión docente:

1.  **Encapsulamiento de Datos**: Las preguntas y opciones están embebidas en el código, permitiendo la ejecución offline por parte del alumno si es necesario.
2.  **Sistema Anti-Copia**: Implementación de bloqueos de teclado y acciones de navegador para garantizar la integridad de la prueba en aula virtual.
3.  **Generación Dinámica de Resultados**: Al finalizar, el motor JavaScript procesa las respuestas y genera automáticamente un archivo **CSV** descargable para el alumno.

## 🔄 El Ciclo de Corrección

Este repositorio es la fuente de datos para el ecosistema de evaluación. El flujo de trabajo completo es:

1.  El alumno realiza el test en estos **HTML**.
2.  El sistema genera un **CSV** con las respuestas.
3.  El docente procesa los archivos masivamente usando el [**Java Exam Automator**](https://github.com/heipry/corrector-examenes) para obtener notas y estadísticas instantáneas.
